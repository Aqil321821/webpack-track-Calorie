# 🥗 Track Calorie App

A simple **Calorie Tracking Web Application** built with **JavaScript (OOP principles)** and bundled using **Webpack**.  
This app allows users to track meals, workouts, and daily calorie limits while storing data in the browser using **Local Storage**.

---

## 🚀 Features

- Set a **daily calorie limit**
- Add **meals** with calories
- Add **workouts** with calories burned
- Automatically calculate **remaining calories**
- Delete individual meals or workouts
- Reset the entire day
- Data persistence using **LocalStorage**
- Organized code using **JavaScript Modules**
- Built with **Object-Oriented Programming (OOP)**
- Bundled and optimized using **Webpack**

---

## 🛠 Tech Stack

- **JavaScript (ES6+)**
- **HTML5**
- **CSS3**
- **Webpack**
- **Node.js**
- **NPM**
- **LocalStorage API**

---

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Aqil321821/webpack-track-Calorie.git

cd  webpack-track-Calorie

npm install

```

## Run Development Server

```bash
 npm run dev
```

## Build for Production

```bash
npm run build
```

Webpack will generate optimized production files inside the dist folder.
